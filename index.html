ПЛАНЕР
    • ВХОДЯЩИЕ
    • Входящие
      • все быстрые дела, многоэтапные проекты, письма, документы, справочные материалы и задачи на будущее, которые надо добавить в планер, чтобы обработать позже
    • Актуальные
      • 1.быстрые задачи - выполнение занимает около 2 минут;
      2.проекты - долгосрочные, делятся на несколько этапов - физических действий.
    • Отложенные
      • 1.справочные материалы, которые надо сохранить - инструкции, адреса, визитные карточки и т.п. 2.лист ожидания - отложенные на неопределённый период проекты и задачи.

      • Дом

      • Здоровье

      • Уход за собой
      • общая тематика для разных проектов - одна из сфер жизни. Например, работа, дом, учёба, личное, хозяйство и т.п.
    • Контексты
      • Сортировка по контекстам
        • Телефон
        • Вне дома
        • Школа
        • Дет. сад
        • Компьютер
        • Минск
        • В городе
        • Делегировано
      • общее место выполнения или один и тот жеинструмент для выполнения разных задач (этапов из разных проектов). Например, телефон, вне дома, магазин, компьютер и т.п.
    • Чек-листы
      • список пунктов для рутины или для какого-то повторяющегося дела. Например, список утренней рутины для сбора на работу или список повторяющихся действий для подготовки к запуску нового проекта.
